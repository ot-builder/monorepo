## Font Merging

### Function <Fn s={CliProc.mergeFont(extend("GS1", Ot.GlyphStore),extend("GS2", Ot.GlyphStore))} args={{basis:Ot.Font("GS1"),override:Ot.Font("GS2"),options:CliProc.MergeOptions,gsf:Ot.GlyphstoreFactory("GS1")}}/>

Create a combined font _override_ into font _basis_, which will have a new glyph store and new `cmap`, `GSUB`, `GPOS` and `GDEF` tables. The remaining tables will be shared with _basis_.
