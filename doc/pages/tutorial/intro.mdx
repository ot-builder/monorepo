`ot-builder` is a Node.JS library that focuses parsing, manipulating and building fonts.

Currently `ot-builder` supports:

- TrueType and Postscript glyph geometry and hinting;
- OpenType layout (`GSUB`,Â `GPOS`, `GDEF`, `BASE`);
